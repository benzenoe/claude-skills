# Claude Skills

A consolidated collection of Claude Code skills.

## Skills
- **blogpost** - Generates publication-ready blog posts for Eytan.com
- **edmund-bogen-mastermind** - Creates Edmund Bogen branded mastermind invitations and workbooks
- **eytan-social-media** - Automates social media presence across platforms
- **job-hunter** - Searches Luxembourg job boards, matches opportunities, generates cover letters

## Installation
```bash
git clone https://github.com/benzenoe/claude-skills.git ~/.claude/skills
```
